The folder contains the empirical epidemic data for:

1. [wave3.days.nwk](wave3.days.nwk) -- transmission tree of wave 3 of SARS-CoV-2 epidemic in Hong Kong from [Xie _et al._ 2024](https://doi.org/10.1093/molbev/msae232), 
resolved with contact-tracing data and rescaled to days.
2. [HIV_Zurich.nwk](HIV_Zurich.nwk) -- transmission tree of HIV-1B MSM epidemic in Zurich from [Voznica _et al._ 2022](https://doi.org/10.1038/s41467-022-31511-0).

It also contains a script [main.py](main.py) that can be used to run the inference on these datasets with BD(EI)(SS)(-CT) models.
